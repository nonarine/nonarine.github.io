# Keyboard Shortcuts

## Global Shortcuts

**`C`** - Clear screen (when not typing in input)

**`Ctrl+,`** or **`Cmd+,`** - Open Settings modal

**`Escape`** - Close modal

## Mouse Controls

**Drag:** Pan the view

**Scroll:** Zoom towards cursor position

**Pan/Zoom buttons:** Precise view control with directional buttons and zoom in/out
